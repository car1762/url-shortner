# 🔗 URL Shortener

A simple and powerful URL shortener built with **Node.js**, **Express**, **MongoDB**, and **EJS**.

This app allows users to shorten long URLs and track how many times each shortened URL has been visited.

---

## 🚀 Features

- Shorten any valid URL
- Automatically generate a unique `shortId`
- Track total clicks (visit history)
- View all URLs in a table
- Mobile-responsive EJS frontend
- MongoDB used for persistence

---

## 🛠️ Tech Stack

- **Node.js** + **Express** – server & routing
- **MongoDB** + **Mongoose** – database and schema
- **EJS** – for dynamic frontend templating
- **CSS + Media Queries** – responsive styling
